# 🌹 Dark Rose Tattoo Studio

A professional, high-end web application for Dark Rose Tattoo Studio featuring a full-screen video background, metallic typography, and sophisticated animations. Built with React, TypeScript, and Vite.

## ✨ Features

- 🎥 Full-screen autoplay video background with overlay effects
- ✨ Metallic silver gradient typography using Cinzel Decorative font
- 🎨 Responsive navigation menu with smooth animations
- 📱 Mobile-optimized with safe area insets for notched devices
- 🌐 Advanced viewport unit calculations for true full-screen experience
- ⚡ Fast development with Vite HMR
- 🎯 TypeScript for type safety
- ✅ Production-ready build system

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
# Build for production
npm run build
```

The production build will be in the `dist/` directory.

### Preview Production Build

```bash
# Preview the production build locally
npm run preview
```

## 🎨 Customization

The theme uses a dark gothic aesthetic with the following color palette:
- Primary: Dark red/rose (#ff1744, #8b0000)
- Background: Deep blacks (#0d0d0d, #1a0a0f)
- Accents: Rose gradient effects

To customize the theme, edit the CSS variables and colors in:
- [src/index.css](src/index.css) - Global styles
- [src/App.css](src/App.css) - Component styles
